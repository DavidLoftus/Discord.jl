    Julicord.add_handler! — Method.
    add_handler!(c::Client, evt::Type{<:AbstractEvent}, func::Function)

    Add a handler for the given event type. The handler should be a function which takes two arguments: A Client and an AbstractEvent (or a subtype). The handler is appended the event's current handlers.

    Note

    The set of handlers for a given event is stored as a Set{Function}. This protects against adding duplicate handlers, except when you pass an anonymous function. Therefore, it's recommended to define your handler functions beforehand.

    Also note that there is no guarantee on the order in which handlers run.
